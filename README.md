Initializing a Git repository
```git init -b main```

Set Up Git Repository on your Local Device:
```git remote add origin https://github.com/Ben168-stack/CS50_Final_Assignment```

To Check If URL Set Correctly:
```git remote -v```

Pull Repo From Main:
```git pull origin main```
